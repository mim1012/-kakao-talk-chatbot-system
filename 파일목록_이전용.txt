## 4. 환경 설정 (새 컴퓨터에서)

1. Python 3.8+ 설치
2. 가상환경 생성:
   python -m venv venv
   venv\Scripts\activate (Windows)

3. 의존성 설치:
   pip install -r requirements.txt

4. PaddleOCR 추가 설치 (필요시):
   pip install paddlepaddle paddleocr

## 5. 압축 명령어 (현재 컴퓨터에서)

PowerShell에서:
```powershell
# 필수 파일만 압축
Compress-Archive -Path @(
    "main.py",
    "config.json", 
    "requirements.txt",
    "paddleocr_optimization.py",
    "src",
    "tools"
) -DestinationPath "chatbot_system_transfer.zip"
```

또는 전체 압축 (테스트, 문서 포함):
```powershell
Compress-Archive -Path @(
    "main.py",
    "config.json",
    "requirements.txt", 
    "paddleocr_optimization.py",
    "src",
    "tools",
    "tests",
    "docs",
    "README.md"
) -DestinationPath "chatbot_system_full.zip"
```
